# Analise de Produtividade Agricola por Cultura

## 📌 Resumo Executivo

Esta análise investiga a relação entre produção, área plantada e produtividade em culturas agrícolas.

Os resultados mostram uma forte concentração de produção em poucas culturas, além de diferenças relevantes de eficiência produtiva entre elas.

---

## ❓ Pergunta de Negócio

Como a produção agrícola se distribui entre culturas e quais são mais eficientes em termos de produtividade?

---

## 💡 Principais Insights

- Alta concentração de produção em poucas culturas
- Diferença significativa de produtividade entre culturas
- Relação direta entre área plantada e volume produzido
