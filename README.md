# 🌱 Análise de Produtividade Agrícola (Crop Yield Analysis)

## ❓ Pergunta de Negócio

Como a produção agrícola se distribui entre culturas e quais são mais eficientes em termos de uso de área?

---

## 📌 Resumo Executivo

Esta análise investiga a relação entre produção, área plantada e produtividade em diferentes culturas agrícolas.

Os resultados mostram que a produção está altamente concentrada em poucas culturas, enquanto outras apresentam maior eficiência produtiva (produção por área).

Para melhorar a análise, foi necessário tratar outliers (como culturas extremamente dominantes), permitindo uma leitura mais precisa da distribuição agrícola.

---

## ⚙️ Metodologia

* Tratamento e organização dos dados no Power BI
* Análise de três dimensões principais:

  * Produção total
  * Área plantada
  * Produtividade (Production / Area)
* Aplicação de filtros e remoção de outliers para melhor interpretação

---

## 📊 Principais Análises

### 🌾 Produção por Cultura

* Identificação das culturas com maior volume produtivo
* Uso de escala logarítmica para melhor comparação

### 🌱 Área Plantada

* Comparação do uso de recursos por cultura
* Identificação de culturas com grande ocupação territorial

### 📈 Produtividade

* Cálculo de eficiência produtiva
* Destaque para culturas com alta produção relativa à área

---

## 💡 Principais Insights

* A produção agrícola é altamente concentrada em poucas culturas
* Existe um trade-off entre volume de produção e eficiência produtiva
* Culturas com menor área podem apresentar alta produtividade
* A remoção de outliers melhora significativamente a análise comparativa

---

## 🚀 Aplicação no Negócio

Essa análise pode apoiar:

* Planejamento agrícola
* Alocação de recursos
* Tomada de decisão operacional
* Monitoramento de eficiência produtiva
